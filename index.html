Bank Management System

Finance Tracker

This Finance Tracker Application in Java enables users to create profiles, manage expenses, and track income. Users can register with their name, address, mobile number, and annual income, all while ensuring valid inputs. Future improvements will include database integration for persistent user data and enhanced financial analytics.

This Weather Application uses Flask to deliver real-time weather updates for user-inputted cities, fetching data from the OpenWeatherMap API. Future enhancements could include saving favorite locations. This project highlights practical web development and API integration skills.
