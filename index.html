Bank Management System

This Bank Management System in C++ allows users to create accounts, deposit and withdraw funds, check balances, and update account information. It features robust input validation to ensure data integrity. The program operates through a user-friendly menu interface. Future enhancements will include integration with a MySQL database for persistent data storage and improved transaction management.

Finance Tracker

This Finance Tracker Application in Java enables users to create profiles, manage expenses, and track income. Users can register with their name, address, mobile number, and annual income, all while ensuring valid inputs. The app features a simple menu for creating users, selecting existing profiles, and managing finances. Future improvements will include database integration for persistent user data and enhanced financial analytics.

Weather App

This Weather Application uses Flask to deliver real-time weather updates for user-inputted cities, fetching data from the OpenWeatherMap API. It defaults to Toronto if no input is given and includes error handling for invalid cities. With a responsive design, it ensures accessibility across devices. Future enhancements could include saving favorite locations. This project highlights practical web development and API integration skills.
